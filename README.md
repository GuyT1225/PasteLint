# pastelint
Privacy-first text tools. Clean messy text. Humanize AI writing. Browser-only.

## 🧠 Product Philosophy

PasteLint is built to feel like a modern AI tool — without requiring AI.

We prioritize:
- Speed (instant, browser-based)
- Privacy (no uploads, no tracking)
- Clarity (simple, predictable output)
- Trust (transparent transformations)

Instead of opaque “magic,” PasteLint aims to make text improvements visible and understandable.

---

## 🚀 SecondDraft (Core Tool)

SecondDraft rewrites AI-sounding or overly formal text into natural, human-readable language.

Key features:
- Sentence-level rewriting
- Tone modes (Natural, Concise, Professional, Friendly, Direct)
- Multi-version output
- Quick refinement actions
- Character and word tracking

---

## 🔬 Upcoming Enhancements

We are actively evolving SecondDraft into a more intelligent writing assistant:

### ✨ Explainability Layer
- “What changed” panel
- Highlights removed filler phrases and simplified language
- Builds trust through visible transformations

### 🎛 Rewrite Strength Control
- Adjustable rewrite intensity (Light → Strong)
- Prevents over-editing while giving users control

### 🧠 Humanization Mode
- Designed specifically for AI-generated text
- Breaks overly perfect structure
- Introduces natural variation and flow
